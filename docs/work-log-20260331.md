# 作業ログ: my-shiba 改善 (2026-03-31)

## 背景

Slack デスクトップアプリの仕様変更により、my-shiba が投稿する草画像（GitHub contribution graph）の右端が見切れるようになった。最新のコントリビューションが確認できない状態だった。

## 実施内容

### 1. Slack 画像表示の修正

**問題**: 画像のアスペクト比が横長すぎ（約5:1）で、Slack のインラインプレビューに収まらない

**対応**:
- PNG 出力幅を 1500px → 1200px に変更
- 縦パディングを 15px → 50px に増やし、アスペクト比を ~3.4:1 に改善
- `image block + slack_file` 方式も試したが `invalid_blocks` エラーで断念。ファイル共有前に `slack_file` は参照できないため
- 最終的に `files.completeUploadExternal` + `channel_id` の従来方式 + 幅/パディング調整で解決

**関連コミット**:
- `b055e67` fix: Use image block for Slack posting (→ 失敗、後に revert)
- `cede221` fix: Revert to completeUploadExternal with channel sharing
- `88a36f8` fix: Increase vertical padding to improve Slack inline preview aspect ratio

### 2. 月/曜日ラベルの表示

**問題**: ghchart API の SVG には月ラベル（Apr〜Mar）と曜日ラベル（Sun〜Sat）が含まれているが、一部が `display:none` で非表示。さらに Cloudflare Workers にはシステムフォントがないため、resvg-wasm がテキストをレンダリングできない

**対応**:
- SVG 内の `display:none` を除去して全ラベルを表示
- Inter フォント（Regular, 402KB）を bundle し resvg に読み込ませる
- Free プランのバンドルサイズ制限（gzip 1MB）超過のため、`pyftsubset` で英数字のみにサブセット化（402KB → 21KB）
- wrangler の `[[rules]]` で `.ttf` を `Data` 型として import
- `fontBuffers` には `ArrayBuffer` ではなく `new Uint8Array(interFont)` を渡す必要があった（2回のデプロイ失敗で判明）

**関連コミット**:
- `bc93b07` feat: Show day/month labels, uniform padding, post to channel not thread
- `35cef44` feat: Bundle Inter font for SVG text rendering in Cloudflare Workers
- `f49276e` fix: Subset Inter font to 21KB for Cloudflare Workers free plan limit
- `175711a` fix: Pass font buffer directly to resvg instead of fetching
- `6726912` fix: Convert font ArrayBuffer to Uint8Array for resvg fontBuffers

### 3. スレッド投稿の廃止

**変更**: メンション返信をスレッドではなくチャンネルに直接投稿するように変更。`threadTs` パラメータを削除。

**関連コミット**: `bc93b07`

**関連コミット**: `b055e67`

## CI トラブルシューティング

| 問題 | 原因 | 解決 |
|------|------|------|
| `npm ci` で esbuild Missing | macOS の lock ファイルに Linux 用 optional deps が不足 | lock ファイル再生成 |
| 再生成後も esbuild Invalid | vitest v4 (vite 8) が esbuild ^0.27 を要求、wrangler 3 は 0.17 | wrangler v4 へアップデートで解消 |
| `invalid_blocks` エラー | `slack_file` を image block で使うにはファイルが事前共有されている必要 | `completeUploadExternal` + `channel_id` 方式に revert |
| `Invalid URL: [object ArrayBuffer]` | TTF import が ArrayBuffer なのに `fetch()` に渡していた | 直接 `fontBuffers` に渡す |
| `[object ArrayBuffer]` エラー | resvg の `fontBuffers` は `Uint8Array` を期待 | `new Uint8Array(interFont)` に変換 |
